# Scientific Infographic: ANN Algorithms Comparison

## ZONES
- Top: Title "近似最近邻搜索算法"
- Three columns comparing IVF, HNSW, LSH
- Bottom: Selection guide

## LABELS
IVF (Inverted File Index):
- Clustering-based approach
- "图书馆分类" analogy
- Speed: O(log N + K)
- Memory: Medium

HNSW (Hierarchical Navigable Small World):
- Multi-layer navigation graph
- "高速公路" analogy (highways + streets)
- Speed: O(log N)
- Memory: High

LSH (Locality Sensitive Hashing):
- Hash-based approach
- Probability-based collision
- Speed: O(1)
- Memory: Low

Best choice callout: HNSW
